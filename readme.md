# Usage Guide:

## To start the Virtual Environment:

Windows (Command Prompt): `.venv\scripts\activate`

Mac/Linux (Terminal): `source .venv\scripts\activate.ps1`

## To start the server on localhost:

Run `python manage.py runserver`
